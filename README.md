# Synara

Synara is a minimal web GUI for coding agents (currently Claude Code, Codex, Devin CLI, Gemini, Kilo Code, OpenCode, more coming soon).

This project started as a clone of [T3Code](https://github.com/pingdotgg/t3code), and has since been customized into its own product with different branding, packaging, release wiring, and product-level behavior.

![Synara screenshot](./assets/prod/readme-screenshot.png)

## How to use

> [!WARNING]
> Install and authenticate the CLI for the provider you want to use. Devin requires Devin CLI installed and authenticated with `devin auth login`; Synara runs it through `devin acp`, and model availability depends on your Devin/Windsurf account/team settings.

You can also just install the desktop app. It's cooler.

Install the [desktop app from the Releases page](https://github.com/Emanuele-web04/Synara/releases)
